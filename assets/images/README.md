Папка assets/images — сюда загружайте фотографии и изображения для сайта.

Рекомендации по использованию:
- Имена файлов: латиница, без пробелов, дефис как разделитель, например: lipovy-2025.jpg
- Форматы: предпочтительно JPEG для фото (.jpg/.jpeg) и PNG для графики с прозрачностью (.png). Можно WebP для экономии места.
- Размеры: для героев/широких блоков — ширина ~1600–2400px; для карточек — ~800px по ширине; миниатюры — ~400px.
- Оптимизация: сжимайте изображения перед загрузкой (например, ImageOptim или онлайн‑сервисы). Поддержите разумное качество (70–85%).
- Организация: можно создавать подпапки, например: `assets/images/products/`, `assets/images/hero/`.

Как ссылаться в HTML:
- Прямой путь от корня проекта: `<img src="assets/images/products/lipovy.jpg" alt="Липовый мёд">`.
- Для CSS: `background-image: url("../assets/images/hero/field.jpg");` (путь зависит от местоположения CSS файла).

Примечание: в репозитории добавлен файл .gitkeep, чтобы папка сохранялась в системе контроля версий до загрузки реальных изображений.
