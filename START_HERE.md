# 🚀 Быстрый старт

## Что было сделано?

Реализован **вертикальный CSS Scroll Snap** для мобильной версии сайта "Пасека".

- **На мобильных (≤768px):** две секции "прилипают" как отдельные экраны при скролле (100vh каждая)
- **На десктопе (>768px):** обычный скролл, snap отключен
- **Без JavaScript:** только CSS
- **Все требования выполнены:** ✅

---

## Как это выглядит?

```
Мобильный (375px)          Десктоп (1920px)
────────────────────────   ────────────────────────

Экран 1:                   ┌─── Header ───────────┐
┌──────────────────┐       │                      │
│ Header           │       │ Hero (2-column)      │
│ Hero текст       │       │                      │
│ [Image]          │       ├──────────────────────┤
└──────────────────┘       │ Рекомендуем          │
↓ SNAP ↓                   │ [Cards grid 3x3]     │
Экран 2:                   │ © Footer             │
┌──────────────────┐       └──────────────────────┘
│ Рекомендуем      │
│ [Cards scroll]   │       ← Обычный скролл
│ © Footer         │          Snap отключен
└──────────────────┘
↑ SNAP ↑
```

## Файлы, которые изменены

### 1. `index.html`
Новая структура:
```html
<div class="hero-wrapper">
  <section class="hero-section hero-section--top">
    <!-- Header + Hero -->
  </section>
  <section class="hero-section hero-section--bottom">
    <!-- Рекомендуем + Footer -->
  </section>
</div>
```

### 2. `styles.css`
Добавлен блок (≤768px):
```css
@media (max-width:768px){
  .hero-wrapper {
    scroll-snap-type: y mandatory;
  }
  .hero-section {
    height: 100vh;
    scroll-snap-align: start;
    scroll-snap-stop: always;
  }
  /* ... остальной CSS ... */
}
```
