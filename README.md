Проект: Пасека — минималистичный фронтенд магазина мёда

Описание
- HTML5, CSS3 (Grid + Flexbox), Vanilla JS
- Страницы: index.html, catalog.html, about.html, cart.html
- Логика корзины: `cart.js` сохраняет в localStorage, предоставляет глобальный объект `window.Cart`
- Поиск/рендер каталога: `catalog.js` с мок-данными

Как запустить
1. Откройте файлы напрямую в браузере: откройте `index.html` или `catalog.html` в локальном браузере.
2. Для разработки удобнее поднять простой статический сервер, например:

```bash
# macOS / Linux
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

Короткий список функций
- Рекомендованные товары на главной странице.
- Страница каталога с поиском (debounce 220ms).
- Добавление товара в корзину — глобальный Cart сохраняет состояние в localStorage.
- Страница корзины с возможностью изменить количество и оформить заказ (демо).

Файлы
- index.html — главная
- catalog.html — каталог + поиск
- about.html — страница "О нас"
- cart.html — корзина и оформление
- styles.css — стили
- catalog.js — рендер товаров и поиск
- cart.js — Cart API и привязки UI
