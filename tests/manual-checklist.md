Manual checklist — тесты для QA

1) Добавление товара
- Откройте `catalog.html`, нажмите «Купить» у товара.
- Проверьте: число в шапке (`#cart-count`) увеличилось.
- Откройте `cart.html` — товар должен быть в таблице.

2) Перезагрузка и persistence
- Добавьте товар, перезагрузите страницу. Корзина должна восстановиться.

3) Обновление количества и удаление
- На странице `cart.html` поменяйте значение в поле количества на 0 — товар удалится.
- Измените значение на 2 — итоговая сумма пересчитывается.

4) Поиск
- На `catalog.html` введите «Липовый» (регистр не важен) — результат должен содержать соответствующий товар.
- Введите случайную строку — появится пустое состояние.

5) Оформление заказа (демо)
- Заполните форму оформления, нажмите «Оформить заказ» — корзина очистится.

6) Синхронизация между вкладками
- Откройте `catalog.html` и `cart.html` в разных вкладках. Добавляйте товар в одной вкладке — вторая должна обновиться (при storage-событии).

7) Адаптив
- Проверьте отображение на экранах 1200px, 800px, 360px.

8) Accessibility
- Убедитесь, что можно управлять формой и кнопками с клавиатуры.
